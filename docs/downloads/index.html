Docker

The fastest way to run Dilla. Images are published to GitHub Container Registry.

# Latest stable release
docker pull ghcr.io/dilla-chat/dilla-chat:latest

# Latest dev build (from main branch)
docker pull ghcr.io/dilla-chat/dilla-chat:dev

# Run
docker run -p 8080:8080 -v dilla-data:/app/data ghcr.io/dilla-chat/dilla-chat:latest
